# TWRP-protect
Script to password protect a TWRP recovery image

Usage:

[Linux]   ./protect.py <recovery.img>

[Windows] python protect.py <recovery.img>

Python3 is required.

Tested on Linux and Windows.

Please report any issues you might find.
